# Analysis Archive

This directory contains historical and archived analysis assets. Current outputs remain in `analysis/results/research_questions/` and `analysis/images/research_questions/`.

## Structure

- `data/csv/`: archived CSV datasets, listings, Supabase exports, and research-question snapshots.
- `data/txt/`: archived loose TXT notes and summaries that are not part of the active analysis output.
- `legacy/outputs_2026-06-27/csv/`: legacy tabular outputs from the 2026-06-27 analysis pass.
- `legacy/outputs_2026-06-27/txt/`: legacy text summaries from the 2026-06-27 analysis pass.
- `legacy/figures_2026-06-27/`: legacy exploratory, presentation, and validation figures.
- `legacy/scripts_2026-06-27/`: legacy R scripts retained for auditability.
