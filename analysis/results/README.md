# Research Results

## Active Analysis

The active statistical outputs are in:

- `analysis/results/research_questions/`

The active input is:

- `analysis/data/juiciness_clean_dataset.csv`

They are generated by:

```bash
Rscript analysis/research_question_analysis.R
```

## Order

1. `juiciness_clean_dataset.csv`: clean dataset emitted for reproducibility.
2. `juiciness_clean_quality_flags.csv`: retained quality audit flags.
3. `juiciness_clean_removed.csv`: rows removed from analysis; currently empty by design.
4. `01_assumption_checks.csv`: Shapiro-Wilk, Bartlett and Brown-Forsythe checks.
5. `02_rmd_style_model_sequence.csv`: model sequence matching the attached Rmd logic.
6. `03_factorial_anova.csv`: factorial ANOVA table.
7. `04_art_factorial_sensitivity.csv`: ART nonparametric sensitivity checks.
8. `05_interaction_emmeans_followup.csv`: post-hoc follow-up only for detected interactions.
9. `06_count_models_duration_offset.csv`: count models for event totals with duration offset.
10. `07_rq3_isolated_vs_control.csv`: isolated treatments compared with Control.
11. `08_rq3_giq_performance_spearman.csv`: GIQ-performance trade-off correlations.
12. `09_rq1_factor_mean_differences.csv`: derived RQ1 factor summaries.
13. `10_rq2_factor_mean_differences.csv`: derived RQ2 factor summaries.
14. `summary.txt`: text summary for quick review in RStudio.
