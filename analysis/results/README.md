# Research Results

## Active Analysis

The active statistical outputs are in:

- `analysis/results/research_questions/`

They are generated by:

```bash
Rscript analysis/research_question_analysis.R
```

## Order

1. `00_hard_clean_*`: data-cleaning audit and final Hard Clean dataset.
2. `01_assumption_checks.csv`: Shapiro-Wilk, Bartlett and Brown-Forsythe checks.
3. `02_rmd_style_model_sequence.csv`: model sequence matching the attached Rmd logic.
4. `03_factorial_anova.csv`: factorial ANOVA table.
5. `04_art_factorial_sensitivity.csv`: ART nonparametric sensitivity checks.
6. `05_interaction_emmeans_followup.csv`: post-hoc follow-up only for detected interactions.
7. `06_count_models_duration_offset.csv`: count models for event totals with duration offset.
8. `07_rq3_isolated_vs_control.csv`: isolated treatments compared with Control.
9. `08_rq3_giq_performance_spearman.csv`: GIQ-performance trade-off correlations.
