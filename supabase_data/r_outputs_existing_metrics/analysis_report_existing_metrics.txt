
Input
=====
Input directory: supabase_data
Output directory: supabase_data/r_outputs_existing_metrics

Guarantee
=========
This analysis uses only variables measured by juicy-vs: condition flags, Run summaries, DamageTaken events, GameSnapshot telemetry, and optional RunFeedback responses if present. Upgrade tables and unmeasured metrics such as task_accuracy, reaction_time_ms, aim_shots, and aim_hits are intentionally excluded.
